Dependensee

Github

Dependensee is an open source patch/vulnerability manager for third party libraries.
It can keep track of libraries used in your projects and will report any CVEs those libraries are vulnerable to, providing links, CVE information, reports and PoC exploits where avaliable. Dependency files can be uploaded by either the API, or by using the UI.
